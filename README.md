# Car_dealership
this is a car dealership program, here you can see the information and filter any of the cars in a database.
i created a connection with my program using XAMPP, mysql and phpMyAdmin, just a simple database with car information.
then i used streams and lambdas to create the filters for my program.

i downloaded de connector for mysql and included it in my project with this link:
[MySQL Community Downloads](https://gemini.google.com/app/bac740ebdf681b30#:~:text=MySQL%20Community%20Downloads) in "platform independient"

I used IntelliJDEA for this project, where i used functional programming, OOP and databases.
